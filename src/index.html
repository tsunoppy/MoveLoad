How to use "MoveLoad".

Welcome to "MoveLoad" to calculate the stress of the simple beam model due to moving loads for the crane.

General, Menu Bar.

- File

New File: Reload window
Open: Open the input file
Save: Save the current input
Print: print the page of "Analysis" tab
Close: Close current window
Exit: Exit application

- Edit

Cut: cutting window
Copy: copy window
Paste: paste text from copy

- View

Changing the window size.

- Tools

Developer mode can check the souse code.

- Help

Copyright, Contact

Input Data

- Geometry

Span: Span of the single beam
Young Mod. Young's Modulus of the girder.
Inertia Inertia of the girder.

- Loads

ω Distributed load, such as self weight.
Load; Wheel Load
Spacing Spacing dimension from the one before.
Impact Factor. Impact factor multiplying wheel load.

- General

Title; Title of the analysis
Coments by HTML souse Description of the analysis
Data Control Maximum figure, when making the graph.
